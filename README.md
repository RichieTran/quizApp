# Quizzr

Quizzr is a CLI Python Quiz App that quizzes users on Python topics. It features a local login system, tracks scores and performance statistics securely, allows feedback on questions to influence future selections, and includes a streak tracker for consecutive correct answers.

## How to Run

1. Ensure you have Python installed.
2. Run `python main.py` from the project directory.
3. Follow the prompts to log in or create a new account.
4. Select a topic.
5. Answer the questions, provide feedback, and view your results.

## Features

- Local user authentication with hashed passwords.
- Question bank loaded from `questions.json`.
- Topics selection based on question categories.
- Score tracking and statistics saved securely.
- Feedback system to influence question selection.
- Streak counter for consecutive correct answers.
- Error handling for missing files, invalid inputs, and interruptions.
